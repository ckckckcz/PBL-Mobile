# Asset Lottie Animation

Folder ini digunakan untuk menyimpan file animasi Lottie (.json).

## Animasi yang Direkomendasikan

Anda dapat mendownload animasi Lottie gratis dari [LottieFiles](https://lottiefiles.com) dengan tema lingkungan:

### Animasi yang Cocok:
1. **Recycling Animation** - Untuk loading saat deteksi
   - Keyword: "recycling", "eco", "environment"
   - URL: https://lottiefiles.com/search?q=recycling&category=animations

2. **Earth/Planet Animation** - Untuk splash screen atau welcome
   - Keyword: "earth", "planet", "save earth"
   - URL: https://lottiefiles.com/search?q=earth&category=animations

3. **Plant Growing** - Untuk success message
   - Keyword: "plant growing", "tree"
   - URL: https://lottiefiles.com/search?q=plant&category=animations

4. **Camera/Scan Animation** - Untuk halaman scan
   - Keyword: "camera", "scan", "photo"
   - URL: https://lottiefiles.com/search?q=camera&category=animations

## Cara Menggunakan:

1. Download file JSON dari LottieFiles
2. Simpan file di folder `assets/lottie/`
3. Gunakan dalam kode:
   ```dart
   Lottie.asset('assets/lottie/nama_file.json')
   ```

## Contoh File yang Bisa Ditambahkan:
- `loading_recycle.json` - Animasi loading
- `success_plant.json` - Animasi sukses
- `earth_rotation.json` - Animasi bumi berputar
- `camera_scan.json` - Animasi kamera

**Note**: File animasi Lottie tidak disertakan dalam repository untuk menghemat ukuran. Silakan download sesuai kebutuhan dari LottieFiles.
